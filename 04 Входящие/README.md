# Входящие

Сюда попадают новые медицинские файлы до переноса в основную базу.

## Поток обработки

1. `00 Новые файлы` — сюда кладутся свежие PDF, фото и текстовые заметки.
2. `10 Черновики AI` — агент складывает первичный разбор.
3. `20 На проверке` — черновики, которые нужно проверить человеку.
4. `30 Одобрено` — пользователь подтвердил разбор, можно переносить в основную базу.
5. `90 Обработано` — исходники и черновики после успешного переноса.
6. `99 Ошибки` — файлы, которые не удалось надежно распознать.

## Правило безопасности

Агент не должен писать финальные медицинские события напрямую в `01 Члены семьи` без явного подтверждения. Сначала создается черновик со статусом `needs_review`.
