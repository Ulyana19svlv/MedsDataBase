# Справочники

Эта папка задает стабильные словари для будущих агентов и для ручной проверки базы. Справочники не являются медицинскими событиями и не должны отображаться в таймлайне как визиты или анализы.

## Файлы

- `people.json` — члены семьи, их стабильные `person_id`, алиасы и ссылки на профили.
- `specialties.json` — нормализованные медицинские направления.
- `document_types.json` — типы входящих документов.
- `metric_dictionary.json` — словарь показателей для будущего слоя динамики анализов.
- `agent_contract.json` — правила, по которым агент должен разбирать новые файлы и создавать черновики.

## Принцип

Человек редактирует Obsidian как обычно. Агент использует эти справочники, чтобы не выдумывать новые названия направлений, типов документов и показателей.
