# Задачи

`tasks.json` — будущий явный слой контрольных задач.

Сейчас дашборд уже умеет вычислять задачи из `follow_up_date` в медицинских событиях. Этот файл нужен для следующего шага: чтобы агент мог создавать отдельные задачи, связанные с документами, событиями и зонами наблюдения.

## Примеры задач

- контроль липидограммы;
- повторить УЗИ;
- записаться к врачу;
- взять документ на прием;
- проверить, что рекомендация врача выполнена.
