# Наблюдение

`watchlist.json` — список тем, которые семья хочет держать на контроле.

Это не диагнозы и не назначения. Это навигационный слой, который помогает агенту отвечать на вопросы и собирать сводки.

## Примеры тем

- липиды / холестерин;
- щитовидная железа;
- беременность и послеродовой период;
- вакцинации и осмотры ребенка;
- аллергии;
- спортивные допуски и нагрузка.
